# Watchdog Video Script – Template

**Title (working):** {COUNTY}: The Line Item That Jumps Off the Page

**Hook (0–8s):** 
“County budget shocker: one department’s spending just spiked **{PCT_CHANGE}%** — and almost nobody noticed.”

**Receipts (on screen):**
- Show the CSV row(s): `{department}` – `{category}` – `{fiscal_year}` vs `{prev_year}`
- Overlay: “Change: {delta_currency} ({PCT_CHANGE}%)”

**Explain (30–60s):**
- “We pulled the official budget into a spreadsheet and ran standard anomaly checks — the same kind auditors use.”
- “This line item breaks its normal pattern by **{zscore}σ**.”
- If monthly: “The spike clusters in **{month_name}** — not seen in prior years.”

**So what? (60–90s):**
- “This could be legit if there was a new project or disaster response — but we didn’t find a matching note in the public docs.”
- “Either way, taxpayers deserve a clear explanation.”

**Call to action (90–120s):**
- “If you work in {COUNTY} finance, drop a comment or send us the memo. We’ll post an update.”
- “Full receipts and data are linked.”
